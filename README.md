# 가짜 리뷰 검사 서비스 (FE)
이 프로젝트는 네이버 블로그 게시글의 가짜 리뷰 여부를 판별하는 서비스의 프론트엔드입니다. 사용자는 검사하고자 하는 블로그 URL을 입력하여 가짜 리뷰 여부를 검사할 수 있으며, 결과를 확인하고 피드백을 남길 수 있습니다.

## 프로젝트 구조

- **프레임워크**: React
- **설치된 라이브러리**:
  - Axios (API 요청 관리)
  - React Router (페이지 라우팅)
  - 기타 UI 라이브러리

## 설치 및 실행

1. **의존성 설치**:
   ```bash
   npm install
   ```
2. **개발 서버 실행**:
   ```bash
   npm start
   ```
개발 서버는 기본적으로 `http://localhost:3000`에서 실행됩니다.
